# Synchorus

여러 핸드폰을 동기화된 스피커로 만드는 Flutter 앱 (P2P).

## 현재 단계
v3 폐루프 리아키텍처 PoC 진행 중 (Android Phase 0~1 완료).
v2(just_audio + 개방 루프)는 legacy로 동작 유지.

## 작업 시작 전
- 설계/결정 기록/진행 상황: **PLAN.md** 필수 확인
  - v3 작업: `핵심 기술 설계 (v3)`, `PoC 플랜`, `7. v3 설계 결정 기록`
  - v2 수정: `핵심 기술 설계 (v2)`, `3-10. 설계 결정 기록`
- 협업 원칙/피드백: memory 참조 (자동 로드됨)

## 기능 수정 후
pubspec.yaml version patch bump (예: 0.0.4+1 → 0.0.5+1). lint/포맷 제외.
